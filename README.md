# Guidez

A free mobile app (iOS & Android) providing resources for people struggling with addiction or mental health issues.

## Features

- **Resource Directory** — Rehab centers, mental health clinics, hospitals, detox centers, and sober living houses searchable by location with map view
- **Authentication** — Account creation, login, and user profiles
- **Social Platform** — Follow users, post content, comment, and direct messaging
- **Trending Page** — Top posts ranked by engagement
- **Data Collection** — Privacy-respecting analytics for personalized support

## Tech Stack

| Layer | Technology |
|---|---|
| Mobile | React Native + Expo |
| Backend / Auth / DB / Realtime | Supabase |
| Facility Data | SAMHSA Treatment Locator API |
| Maps | Google Maps SDK |
| Push Notifications | Expo Notifications + FCM |
| Analytics | Mixpanel |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npx expo start
```
